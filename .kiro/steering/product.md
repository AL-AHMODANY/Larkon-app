# Product

**Larkon** is a Vue 3 admin dashboard template. It provides a full-featured UI shell with:

- Authentication (login / signup with localStorage persistence)
- Sidebar navigation with collapsible sections and icon-only condensed mode
- Topbar with live clock, dark/light theme toggle, notifications, messages, activity stream, language switcher, and user menu
- Theme settings offcanvas (color scheme, topbar color, menu color, sidebar size, auto-logout timer)
- Component showcase pages: Base UI, Advanced UI, Charts, Forms, Tables, Icons, Maps, Menu examples
- Dark / light mode driven by `data-theme` on `<html>` and `data-menu` / `data-topbar` on `<body>`
- Settings persisted to `localStorage` under the key `larkon_settings`

The app is a single-page application with no backend — all state is client-side.
